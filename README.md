# FALLMINE

FALLMINE es un juego de puzzle que combina Tetris con Buscaminas invertido. Fue desarrollado por Lean (Martino) para experimentar con mecánicas híbridas.

## Descripción
El tablero de Tetris es la base donde caen piezas (tetrominós). Cada bloque de cada pieza puede contener una bomba latente. Las bombas no explotan al instante; se detonan al completar una línea, al recibir la explosión de una bomba adyacente o al activar el detonador manual (tecla Espacio). Tras una explosión se revelan números al estilo Buscaminas, indicando cuántas bombas latentes rodean ese bloque.

## Cómo jugar
- Mueve las piezas con las flechas (izquierda/derecha), rota con la flecha arriba y acelera la caída con la flecha abajo.
- Pulsa **Espacio** para detonar manualmente bombas adyacentes a la pieza activa (tiene tiempo de recarga).
- Pulsa **C** para hacer hold y **P** para pausar.
- Pulsa **R** para reiniciar la partida.
- Consulta el puntaje y nivel en el panel lateral.
- No destruyas bloques numerados innecesariamente: conservan información valiosa.

Para probar el juego, abre el archivo `fallmine.html` en tu navegador favorito.
